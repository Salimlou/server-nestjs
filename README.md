# NestJS - Prisma - Apollo 2

* For using Prisma you should signup to https://app.prisma.io/ and deploy your server on it and the replace the end point in .env file

## Description

Motivation to use NestJS

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```
